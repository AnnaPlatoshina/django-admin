Система управления сотрудниками и рабочими местами
Django-приложение для управления информацией о сотрудниках, их навыках и рабочих местах.

Возможности
📋 Управление сотрудниками с расширенными профилями

🎯 Навыки сотрудников с уровнями владения (1-10)

🪑 Рабочие места с привязкой к сотрудникам

🎨 WYSIWYG-редактор для описаний сотрудников

⚡ Панель администратора Django для управления данными

Технологии
Python 3.13

Django 5.2.5

Django CKEditor 5

SQLite (для разработки)

Установка и запуск
Клонируйте репозиторий

bash
git clone <repository-url>
cd django_company
Создайте виртуальное окружение

bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
Установите зависимости

bash
pip install django django-ckeditor-5 Pillow
Примените миграции

bash
python manage.py makemigrations
python manage.py migrate
Создайте суперпользователя

bash
python manage.py createsuperuser
Запустите сервер

bash
python manage.py runserver
Откройте в браузере

text
http://127.0.0.1:8000/admin/
Структура проекта
Модели
Сотрудники (employees)
CustomUser - расширенная модель пользователя

Пол, имя, фамилия, отчество

Должность, отдел

WYSIWYG-описание

Дата приема на работу

Skill - навыки (фронтенд, бэкенд и т.д.)

EmployeeSkill - связь сотрудник-навык с уровнем владения

Рабочие места (workplaces)
Workplace - рабочее место

Номер стола

Связь один-к-одному с сотрудником

Дополнительная информация

Административная панель
Доступна по адресу /admin/ после входа суперпользователя:

Сотрудники - управление пользователями с inline-навыками

Навыки - создание и редактирование навыков

Рабочие места - назначение столов сотрудникам

Использование
Добавьте навыки в разделе "Skills"

Создайте сотрудников в разделе "Custom users"

Назначьте навыки через inline-формы в карточке сотрудника

Создайте рабочие места и свяжите их с сотрудниками

Особенности
🤝 Связь один-к-одному между рабочим местом и сотрудником

📊 Уровни навыков от 1 до 10 с валидацией

🎨 Rich-text редактор для описаний

🔍 Поиск и фильтрация в админке

🌍 Поддержка мультиязычности (русский/английский)
