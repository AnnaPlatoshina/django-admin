Система управления сотрудниками компании

Django-приложение для управления информацией о сотрудниках, их навыках и рабочих местах.

## 🚀 Возможности

- 👥 Управление сотрудниками с расширенными профилями
- 🎯 Система навыков с уровнями владения (1-10)
- 🖼️ Галерея изображений сотрудников
- 🪑 Рабочие места с привязкой к сотрудникам
- ⚡ Панель администратора Django
- 🎨 WYSIWYG-редактор для описаний

## 🛠️ Технологии

- Python 3.13
- Django 5.2.5
- Django CKEditor 5
- Bootstrap 5
- SQLite

## 📦 Установка

```bash
# Клонирование репозитория
git clone https://github.com/AnnaPlatoshina/django-admin.git
cd django-admin

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt

# Миграции базы данных
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
